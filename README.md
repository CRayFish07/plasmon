# Plasmon
PCMan BBS Client on Electron

## Development
[pcman.js](https://github.com/pcman-bbs/pcman.js) is not ready on npm yet, please use `npm link pcman` to develop.

`npm install`

`npm start`

## Development with Livereload

`npm run server`
And also
`npm run livereload`

## Build

`npm run build`
